# hankyul-develop

> 한결입찰 템플릿 제작

## Build 설정

``` bash
# 설치
npm install

# 개발실행과 접속주소 localhost:8080
npm run dev

# 생산환경 생성
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
